# 🧢 ㄱㅈㅂ Golf League Manager (YGC)

> **F1 챔피언십 포인트 시스템** 기반 골프 리그 관리 앱

## 🏎️ Features

| 기능 | 정관 조항 | 설명 |
|------|----------|------|
| **F1 포인트** | 제11조 | 1위 25 / 2위 18 / 3위 15 / 4위 12 / 5위 10 / 6위 8 |
| **모자 추적** | 제1조 | 월례회 꼴찌 자동 판별, 보유일수/횟수 랭킹 |
| **카트배** | 제12조 | 평균타수 기반 밸런스 카트 편성 (스네이크 드래프트) |
| **출석 관리** | 제10조 | 3~6, 8~11월 8회 중 5회 참석 체크 |
| **회비/목표** | 제6,9조 | 150만원 납입, 목표타수 달성 시 50만원 환급 |
| **챔피언십** | 제13조 | 연말 포인트 1위 = 챔피언 모자 + 다음해 총무 |

## 🚀 Quick Start

```bash
npm install
npm run dev
```
